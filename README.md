## 介绍

该程序用于调试代码，一般在竞赛及练习使用。

由于主程序为`.bat`批处理脚本，因此只能在Windows系统上运行。

## 使用

`rand.cpp`为数据生成代码，`my.cpp`为待调试代码，`std.cpp`为标程代码，使用时请根据题目要求更改源码并编译，执行`check.bat`。

若要修改文件名，请修改`check.bat`中对应的文件名。
